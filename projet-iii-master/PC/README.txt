==============
PC-application
==============


DESCRIPTION
-----------
PC application used as an administrative tool for the project. Application
allows admin users to visualize the exchange of data between the data engins and
the mobile application as well as the the data stored by the web server.


USAGE
-----
To run the application, stay in PC/ folder and run './bixi-pc' or run 'make run'.


TESTING
-------
To run tests, stay in PC/ folder and run 'make unitest'.
